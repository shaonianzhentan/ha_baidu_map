# 百度地图
在HA里使用的百度地图，支持GPS定位轨迹显示

# 使用方式
```
ha_baidu_map:
  ak: 百度地图【浏览器的浏览器的浏览器的AK密钥】 - 如果不会配置ak就把这行删掉，初次安装空白就刷新一下页面
```

## 在GPSLogger应用里的配置
```

http://IP:8123/ha_baidu_map-location-【定义的key】?entity_id=【实体ID】&latitude=%LAT&longitude=%LON&battery=%BATT&sts=%STARTTIMESTAMP

```
